Contadores de CSS

Uso

Son como variables que el CSS controla mediante reglas para ajustar la apariencia del contenido basado en su ubicación en el contenido. Su uso más tipico es para titulos en una lista enumerada y con jerarquía, por ejemplo.

Se muestra un ejemplo en una lista no ordenada que se transforma en ordenada gracias al contador. Existe un contador que se resetea en ul-ordered y que a cada li toma un nuevo valor de incremento, por eso aparecen los items numerados

  • Verduras
  • Carne de cerdo
  • Bacalao
