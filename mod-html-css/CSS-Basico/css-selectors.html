Selectores de CSS.

Selectores de css de atributos.

Permiten seleccionar elementos HTML en función de sus atributos y/o valores de dichos atributos. En el siguiente ejemplo se toma la imagen y se aplican estilos en función de los atributos que contiene

Pseudo clases en listas.

Ejemplos de las pseudoclases :first-child, :last-child y :nth-child

First-child y last child.

Se marca el primer elemento con el selector :first-child y se aplica un estilo de color verde y al :last-child un color rojo

Hijo n

Se marcan en verde con el selector :nth-child(2n + 3). Esto implica que n va tomando valores así que para 0, 1, 2, 3.. los elementos que aplican el selector son los 3, 5, 7...

Con :nth-last-child(n) se empieza desde el final

Pseudoclase :empty

Se aplica cuando dinamicamente se tiene previsto dejar un elemento sin innerHTML para aplicar propiedades especificas. En el ejemplo se aplica un padding solo cuando el elemento tiene contenido

Aqui hay contenido

El cuadro del medio está afectado por la pseudoclase :empty por eso aplica un padding de 2px solo, porque no tiene contenido. En cambio el siguiente es una demostracion de como quedaría si no tuviera esta pseudoclase. Se mantiene el padding original

Pseudoelementos

Permite dar contenido a partes especificas de los elementos

::first-line y ::first-letter

Los pseudoelementos ::first-line y ::first-letter permiten modificar el estilo de la primera línea y letra de un texto respectivamente
