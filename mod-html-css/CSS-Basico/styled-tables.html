Personalización de bordes y dimensiones

Se tratan como si fueran otro elemento HTML cualquiera con la propiedad border y sus derivadas. El borde para que tenga efecto se debe aplicar en table, th y td para que se visualice por todo.

Pero con esto se añade un borde doble, porque el borde se añade en cada elemento, hay que colapsarlos

La tabla se ajusta al contenido, pero se puede expandir con la propiedad width

El alto de una fila se puede aplicar con height al elemento th o td. Se propaga al resto, ajustando todas las celdas al más exigente
